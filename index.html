SFPD Incident and Crime Reporter

This website provides information on the latest SFPD-reported incidents and crimes in San Francisco. This page updates daily and was created for educational purposes. To provide the information presented here, the SFPD database is queried daily. Additionally, an AI-powered model is used to present predictions on the possible number of future crime rates.
